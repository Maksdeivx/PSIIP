1. Работа с базой данных (PDO)

    Организовано безопасное подключение к БД через интерфейс PDO.

    Все запросы к базе используют подготовленные выражения (prepared statements) для защиты от SQL-инъекций.

    Централизованная конфигурация вынесена в отдельный файл db.php.

2. Динамический каталог продукции

Реализован полноценный модуль управления отображением товаров:

    Отображение контента: Данные о товарах и категориях динамически подгружаются из таблиц products и categories.

    Поиск: Поиск товаров по названию с использованием оператора LIKE.

    Сортировка: Возможность сортировки по цене (возрастание/убывание) и по новизне.

    Постраничная пагинация: Ограничение вывода (по 4 товара на страницу) с использованием SQL-команд LIMIT и OFFSET.

    Фильтрация: Логика фильтрации товаров по ID категории.

3. Корзина и оформление заказа

    Корзина: Реализована на основе PHP-сессий ($_SESSION). Позволяет добавлять товары, хранить их количество и очищать список.

    Оформление заказа: Данные клиента (имя, телефон) вместе с содержимым корзины сохраняются в таблицу orders. Список товаров сохраняется в формате JSON.

4. Система пользователей (Sessions)

    Регистрация и вход: Создана система аккаунтов с хешированием паролей (password_hash).

    Личный кабинет (profile.php): Авторизованные пользователи могут просматривать историю своих заказов, извлеченную из БД по их уникальному user_id.

    Панель администратора (admin.php): Защищенная страница для управления магазином, позволяющая просматривать все поступившие заказы и логи заявок.

5. Обратная связь и логирование

    PHPMailer: Настроена отправка уведомлений на Email через SMTP-протокол (Gmail).

    Работа с файлами: Заявки на консультацию дублируются в текстовый файл textfile.txt (функции fopen, fwrite).

    Техническая информация: Реализован вывод данных о файловой системе (размер файла, дата изменения, список изображений в папке).

6. Работа с Cookies

    Реализовано отслеживание даты первого визита пользователя.

    Механизм персональных промокодов, сохраняющихся в браузере пользователя на определенный срок.
